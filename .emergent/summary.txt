**Key Technical Concepts**
- **Hybrid Mobile App Development:** Used Capacitor to wrap an existing React web application into a native Android APK, enabling access to native device features.
- **Native Plugin Bridge:** Wrote a custom Java Capacitor plugin to create a bridge between JavaScript and the native Android SDK for the H10P's thermal printer.
- **AIDL (Android Interface Definition Language):** Utilized manufacturer-provided AIDL files to communicate with the background printer service on the H10P device.
- **HID Keyboard Wedge:** Architected the terminal scanner logic to work with the H10P's built-in scanner, which emulates a physical keyboard, by creating a global keyboard event listener.
- **Backend Performance Optimization:** Refactored a slow, sequential database query chain into a highly performant parallel operation using , dramatically improving UI responsiveness.
- **Floating-Point Precision:** Made a critical architectural decision to handle floating-point inaccuracies at the presentation layer (frontend) only, preserving full precision in the database and backend for audit and calculation integrity.
