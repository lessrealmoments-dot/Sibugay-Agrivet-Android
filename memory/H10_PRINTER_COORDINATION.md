## Current state (2026-04-05)

### Web layer (JS / React)

- **PrintBridge.js** — Routes print: `Capacitor.isNativePlatform()` → `H10PPrinter.printHtml`, else browser popup path.
- **H10PPrinterPlugin.js** — `registerPlugin('H10PPrinter', { web: fallback })`; must match **Java** `@CapacitorPlugin(name = "H10PPrinter")`.
- **PrintEngine.js** — `generateHtml` / `print` for thermal (58mm) and full page.
- **Call sites** — `TerminalSales.jsx` (2 buttons), `TerminalShell.jsx` (QuickScan 2 buttons), `DocViewerPage.jsx` (`handleReprint`). All should `await PrintBridge.print(...)` and surface errors (e.g. Sonner toast).

### Android layer

- **H10PPrinterPlugin.java** — `printHtml`: wait for AIDL bind → headless WebView HTML→Bitmap → `PrinterInterface` SDK.
- **AIDL** — `PrinterInterface.aidl` → service `recieptservice.com.recieptservice.service.PrinterService`.
- **printer-release.aar** — Senraise SDK in `frontend/android/app/libs/` (gitignored; not in Git).
- **MainActivity.java** — `registerPlugin(H10PPrinterPlugin.class)` before `super.onCreate()`.
- **APK** — Gradle / Android Studio; live URL in `frontend/capacitor.config.ts` loads hosted web app.

## Print flow (after fixes)

```
User taps Print (TerminalSales / TerminalShell / DocViewer)
  → await PrintBridge.print({ type, data, format, businessInfo, ... })
  → Capacitor.isNativePlatform() === true (inside APK)
  → PrintEngine.generateHtml(...) → HTML string
  → H10PPrinter.printHtml({ html, format })  // JS bridge
  → H10PPrinterPlugin.printHtml (Java)
       → Thread "h10p-print-wait":
            waitForPrinterConnection(12000):
              bindPrinterService()
              loop: sleep 100ms until printerConnected && printer != null OR timeout
       → if !ready: call.reject("Printer service not connected. On H10P/Senraise …")
       → else: runOnUiThread → renderHtmlToBitmap → printBitmap / beginWork / endWork
       → call.resolve({ success: true })
```

**Note:** Wait loop uses **100 ms** sleeps (not 300 ms).

## Debugging (if print still fails)

1. **Receipt service installed**

   ```bash
   adb shell pm list packages | findstr reciept
   ```

   Expect: `package:recieptservice.com.recieptservice`

2. **Logs**

   ```bash
   adb logcat | findstr H10PPrinter
   ```

   `bindService returned false` → service package missing or not exported.

3. **Installed app version**

   ```bash
   adb shell dumpsys package com.agribooks.terminal | findstr versionName
   ```

## Rules of thumb for parallel AI work

- Keep **Capacitor plugin name** exactly **`H10PPrinter`** on JS and Java.
- **HTML** passed to native is generated by PrintEngine (inline CSS; QR may load from `api.qrserver.com`).
- **Bitmap width:** 384 px thermal, 576 px `full_page`; **`format`:** `"thermal"` | `"full_page"`.
- Prefer **one commit** that lands web + Android changes together to avoid “half-fixed” installs.
